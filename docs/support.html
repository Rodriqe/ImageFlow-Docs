Primary Support

Open an issue in the docs repository:

Open Support Issue

What to include in your report

  • Device and OS version (macOS/iOS/iPadOS)
  • Input file type and expected output format
  • Steps to reproduce the issue
  • Screenshot or exact error message if available

Privacy reminder

ImageFlow processes files locally on-device. Please avoid uploading private files in issue reports.
