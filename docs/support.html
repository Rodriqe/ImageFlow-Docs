Before You Contact Support

  • Confirm your OS version is supported (macOS 14+, iOS/iPadOS 17+).
  • Include input format, output format, and exact conversion steps.
  • If possible, include the exact error message shown by the app.
  • Avoid sharing private files in public issues.
