Introduction

ImageFlow ("the App") is a native image converter for macOS, iPhone, and iPad developed by Rodrigo Quezada. This Privacy Policy explains our practices regarding data collection when you use ImageFlow.

Data Collection

ImageFlow does not collect, store, transmit, or share any personal data.
  • No user accounts required
  • No analytics or crash-reporting SDKs included
  • No data is sent to any server
  • No tracking or advertising identifiers used
  • No data shared with third parties

How the App Works

All image processing happens entirely on your device:

Photo Library Access (iOS / iPadOS)

The App uses Apple's system photo picker (PHPickerViewController). This picker runs in a separate process managed by iOS — the App never receives direct access to your photo library. Only the specific image you select is provided to the App.

File Access

Images selected via the Files app or Finder are processed locally and immediately. The App copies the file to a secure local directory for processing.

Output Files

Converted images are saved to:

  • iOS / iPadOS: Your device's Documents folder, visible in Files → On My iPhone/iPad → ImageFlow
  • macOS: A folder of your choice (defaults to ~/Downloads)

Files never leave your device unless you explicitly share them.

Permissions

Permission Platform Reason
Photo Library (read-only, via system picker) iOS / iPadOS Lets you select an image from your Photos library to convert
File access iOS / iPadOS, macOS Lets you open image files from the Files app or Finder
Local Documents folder iOS / iPadOS Saves converted images so they appear in the Files app

No camera, microphone, location, contacts, or network access is used.

Children's Privacy

The App does not collect any information from anyone, including children under the age of 13.

Changes to This Policy

If this policy changes, the updated version will be published here with a new effective date.

🖼️ Política de Privacidad

Última actualización: 24 de marzo de 2026

Introducción

ImageFlow ("la App") es un conversor de imágenes nativo para macOS, iPhone e iPad desarrollado por Rodrigo Quezada. Esta Política de Privacidad explica nuestras prácticas respecto a la recopilación de datos cuando utilizas ImageFlow.

Recopilación de datos

ImageFlow no recopila, almacena, transmite ni comparte ningún dato personal.
  • No se requiere cuenta de usuario
  • No se incluyen SDKs de analíticas ni reportes de errores
  • No se envía ningún dato a ningún servidor
  • No se utilizan identificadores de rastreo ni publicidad
  • No se comparte información con terceros

Cómo funciona la App

Todo el procesamiento de imágenes ocurre completamente en tu dispositivo:

Acceso a la Fototeca (iOS / iPadOS)

La App utiliza el selector de fotos del sistema de Apple (PHPickerViewController). Este selector corre en un proceso separado gestionado por iOS — la App nunca recibe acceso directo a tu biblioteca de fotos. Solo la imagen específica que seleccionas es entregada a la App.

Acceso a archivos

Las imágenes seleccionadas desde la app Archivos o Finder son procesadas localmente de inmediato. La App copia el archivo a un directorio local seguro para procesarlo.

Archivos de salida

Las imágenes convertidas se guardan en:

  • iOS / iPadOS: La carpeta Documentos de tu dispositivo, visible en Archivos → En mi iPhone/iPad → ImageFlow
  • macOS: La carpeta que elijas (por defecto ~/Downloads)

Los archivos nunca salen de tu dispositivo a menos que los compartas explícitamente.

Permisos

Permiso Plataforma Motivo
Fototeca (solo lectura, vía selector del sistema) iOS / iPadOS Permite seleccionar una imagen de tu biblioteca para convertirla
Acceso a archivos iOS / iPadOS, macOS Permite abrir archivos de imagen desde la app Archivos o Finder
Carpeta Documents local iOS / iPadOS Guarda las imágenes convertidas para que aparezcan en la app Archivos

No se accede a cámara, micrófono, ubicación, contactos ni red.

Privacidad de menores

La App no recopila ninguna información de nadie, incluyendo menores de 13 años.
